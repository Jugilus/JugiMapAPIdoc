Detailed Description

Cocos2d-x nodes ownership

Jugimap extended classes which wrap Cocos2d-x node objects does not take ownership over them. When deleting a jugimap scene first delete its Cocos2d scene node via Cocos2d director. For deleting sprites or text objects use dedicated layer functions which also remove related Cocos2d-x nodes.
