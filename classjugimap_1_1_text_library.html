Detailed Description

The TextLibrary class defines a storage for text books.

A TextLibrary object takes ownership of TextBook objects added to it.

TextBook * CreateNewBook ( const std::string &  _bookName)

Create a new text book with the given *_bookName* and returns a pointer link.

If a text book with the given name allready exists in this library no book will be created and the function will return a pointer to the existing text book.
